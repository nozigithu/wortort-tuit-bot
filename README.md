# WortOrt–TUIT Lernbot 🇩🇪

Ein interaktiver Telegram-Bot für den Deutschunterricht im Format des **Distance Learning**.  
Der Bot wurde für Studierende auf dem Niveau **A2–B1** entwickelt und verbindet:

- thematischen Wortschatz
- Lesetexte
- Grammatik
- Missionen
- Quizfragen
- Ranking-System

## Projektziel

Der Bot unterstützt Lernende beim schrittweisen Arbeiten an einer Lektion, ohne dass sie ständig überlegen müssen, was als Nächstes zu tun ist.  
Jede Lektion folgt einem klaren Lernweg:

1. Theorie  
2. Wortschatz  
3. Lesetext  
4. Grammatik  
5. Missionen  
6. Quizfragen  
7. Abschluss und Ranking

Dadurch eignet sich der Bot besonders gut für den **Fernunterricht** und für das **selbstständige Lernen**.

## Funktionen des Bots

- vollständige Benutzeroberfläche auf **Deutsch**
- **Reply-Buttons** für Menüfunktionen
- **Inline-Buttons** für Quizantworten
- Schritt-für-Schritt-Lernpfad
- Punkte-System
- Niveau-Anzeige
- Bestenliste / Ranking
- Speicherung des Lernfortschritts über `users.json`
- Anzeige eines Logos beim Start (`logo.png`)
